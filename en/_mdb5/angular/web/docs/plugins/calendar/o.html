MDB calendar is a plugin that allows you to efficiently manage tasks. Thanks to this extension you will be able to easily create new events, manage current events, move existing events between other days, and store all data in an easily readable array.

Note: Read the API tab to find all available options and advanced customization

Basic example

A few predefined events allowing you to see how the plugin looks like.

import { Component } from '@angular/core'; import { MdbCalendarEvent, startOfDay, endOfDay } from 'mdb-angular-calendar'; @Component({ selector: 'app-root', templateUrl: './app.component.html', }) export class AppComponent { events: MdbCalendarEvent[] = [ { id: 'event1', summary: 'JS Conference', startDate: startOfDay(new Date()), endDate: endOfDay(new Date()), color: { background: 'primary', foreground: 'white', }, }, { id: 'event2', summary: 'Vue Meetup', startDate: startOfDay(new Date(new Date().setDate(new Date().getDate() + 1))), endDate: endOfDay(new Date(new Date().setDate(new Date().getDate() + 5))), color: { background: 'success', foreground: 'white', }, }, { id: 'event3', summary: 'Angular Meetup', description: 'Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur', startDate: new Date(new Date(new Date().setDate(new Date().getDate() - 3)).setHours(10,0,0,0)), endDate: new Date(new Date(new Date().setDate(new Date().getDate() + 3)).setHours(14,0,0,0)), color: { background: 'danger', foreground: 'white', }, }, { id: 'event4', summary: 'React Meetup', startDate: startOfDay(new Date(new Date().setDate(new Date().getDate() + 5))), endDate: endOfDay(new Date(new Date().setDate(new Date().getDate() + 8))), color: { background: 'info', foreground: 'white', }, }, { id: 'event5', summary: 'Meeting', startDate: new Date(new Date(new Date().setDate(new Date().getDate() + 1)).setHours(8,0,0,0)), endDate: new Date(new Date(new Date().setDate(new Date().getDate() + 1)).setHours(12,0,0,0)), color: { background: 'primary', foreground: 'white', }, }, { id: 'event6', summary: 'Call', startDate: new Date(new Date(new Date().setDate(new Date().getDate() + 2)).setHours(11,0,0,0)), endDate: new Date(new Date(new Date().setDate(new Date().getDate() + 2)).setHours(14,0,0,0)), color: { background: 'warning', }, } ]; };

Event formats

Calendar events can be added with time in two different formats - 12h or 24h.

Monday first

Set the mondayFirst input to true so that Monday is the first day of the week.
