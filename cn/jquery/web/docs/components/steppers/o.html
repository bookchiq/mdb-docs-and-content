引导步骤（步进）

Bootstrap步进器是将内容显示为由用户里程碑定义的进程的组件。通过按钮将以下步骤分开并连接。

对于各种各样的注册表格，这是一个很好的解决方案，您不想用大量的字段和问题来吓the用户。

步进器可以垂直和水平对齐。

要开始使用动态步进器，请参阅 “入门” 此页面上的标签。

所有步进标记为 MDB Pro组件需要一个简单的附加设置。看一下 “入门” 标签，或按照我们的影片教学 。
