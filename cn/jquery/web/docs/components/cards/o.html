Bootstrap卡片

引导卡是显示具有不同元素，具有特征性阴影，深度和悬停效果的内容的组件。

除了使用卡之外，很难想到一种更好的向用户显示内容的方法。正如您可能已经在几乎所有产品中注意到的那样，一些最大的公司（如Facebook或Google）都非常了解这一点。

卡片可为您提供清晰，干净的内容分类，以及将其呈现给用户的诱人方式。

MDB通过独特的“材料设计”功能（例如轻微的阴影，深度，级联或波浪效果）增强了Bootstrap卡。除此之外，它还通过其他可选动画，社交图标，化身以及本机Bootstrap中不可用的各种其他效果丰富了它。
