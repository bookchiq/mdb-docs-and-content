MDB使用一组功能强大的图标，称为Font Awesome。

一种字体，1480个图标

在一个集合中，Font Awesome是一种与网络相关的动作的象形语言。

无需JavaScript

更少的兼容性问题，因为Font Awesome不需要JavaScript。

无限的可扩展性

可缩放的矢量图形意味着每个图标无论大小都看起来很棒。

免费，如语音

Font Awesome完全免费用于商业用途。

CSS控件

轻松设置图标颜色，大小，阴影以及CSS可能的样式。

非常适合视网膜显示器

字体真棒图标是矢量，这意味着它们在高分辨率显示器上非常漂亮。

易用性

字体很棒爱屏幕阅读器，并帮助您在网络上访问图标。

较大的图标

  • 如果您的图标在顶部和底部被砍掉，请确保您有足够的行高。

堆叠图标

要堆叠多个图标，请使用 fa-stack 要堆叠的2个图标的父HTML元素上的class。然后添加 fa-stack-1x 常规尺寸图标的类，并添加 fa-stack-2x 较大图标的类。正反可以使用 fa-stack-1x 有助于获得剔除效果。您甚至可以在父级上抛出较大的图标类，以进一步控制大小。
