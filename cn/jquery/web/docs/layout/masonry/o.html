引导砌体

Bootstrap砌体是基于列的网格布局。与其他网格布局不同，它没有固定高度的行。基本上，Masonry布局通过减少不必要的间隙来优化网页内部空间的使用。没有这种类型的布局，就需要某些限制来维持布局的结构。

要开始使用石工，请参阅 “入门” 此页面上的标签。

注意： 砌体集成从版本开始可用 4.8.0 （发布于2019年4月29日）。
