引导移动触摸手势

引导移动手势是与移动设备的基于触摸的交互。各种不同的手指运动指示特定的组件响应。

MDB为您提供最流行的手势支持。

带手势支持的轮播

Bootstrap允许使用触摸支持。它设置为真正默认情况下。

如果您想了解有关轮播的更多信息，请参见我们的文件资料 。

单击下面的按钮以启动实时示例，并查看带有手势支持的轮播在行动。

实时预览
$（&#39;。carousel&#39;）。carousel（{touch：true //默认}）

这是触摸支持设置为假 。触控支持不管用接着就，随即。

实时预览
$（&#39;。carousel&#39;）。carousel（{touch：false}）
