搜索

引导搜索

Bootstrap搜索是使用户能够在文档，网页或其他来源的集合中查找单词，句子和数字的组件。

可以使用按钮或图标来实现，而不是将其放置为输入或导航栏中，以获得更好的用户体验。

Bootstrap搜索示例：

  • 数据库
  • 搜尋引擎
  • 文档页面上的内置搜索框（类似于左侧的搜索框）

您可以使用材料设计版本要么默认的Bootstrap样式 。
