多选MDB Pro组件

自举多重选择

Bootstrap Multiselect是一个基于jQuery的插件，允许用户从标准Bootstrap选择中打钩多个选项。

它的实现非常简单，作为交换，它带来了很多UX价值。

Bootstrap Multiselect用法示例：

  • 披萨递送系统中的配料选择
  • 笔记本电脑硬件配置在线商店
  • 机票预订定制

覆盖标准选择

MDB会覆盖标准选择，以将其替换为我们的材料选择。这就是为什么在使用它之前，必须通过在项目中放置以下代码来初始化Material Select的原因。

此外，您将需要为页面生成的任何动态生成的选择元素单独调用。

初始化
