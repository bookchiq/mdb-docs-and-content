文本域

引导文字区

Bootstrap文本区域是专用于大量文本的输入。它可以在各种组件中使用，例如表单，评论部分和论坛。

Textareas不必很无聊。可以使用颜色，阴影或圆角来增强它们。

Bootstrap textarea使用示例：

  • 评论部分
  • 表格
  • 设计师技术交流区
