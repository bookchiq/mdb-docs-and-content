SideNav MDB Pro组件

Bootstrap Sidenav

Bootstrap sidenav是一种垂直导航组件，除了传统的文本链接之外，还可以嵌入图标，下拉菜单，头像或搜索表单。

由于其清晰和简单，它显着提高了用户体验。它使您可以快速浏览小型应用程序和大型门户。多重链接嵌入功能使您能够实施更高级的内容分类，这在大型项目中几乎是必不可少的。

借助MDB，您可以通过使用各种诱人的侧边菜单之一轻松地在自己的项目中实现SideNav。

左侧导航是SideNav的实时演示。

滑出按钮

滑出按钮是一个触发器，可以在隐藏时启动SideNav。您可以将其放置在任意位置-您只需要添加 .button-collapse 类和 data-activates =“滑出” 所选元素的属性。

选项

// SideNav默认选项$（&#39;。button-collapse&#39;）。sideNav（{edge：&#39;left&#39;，//选择水平原点closeOnClick：false，//关闭侧面导航<a>单击，对Angular / Meteor断点有用：1440 //按钮折叠的断点MENU_WIDTH：240，// sidenav timeDurationOpen的宽度：300，//持续时间打开菜单timeDurationClose：200，//持续时间打开菜单timeDurationOverlayOpen：50，/ /持续时间打开叠加timeDurationOverlayClose：200，//持续时间关闭叠加easingOpen：&#39;easeOutQuad&#39;，//打开动画easingClose：&#39;easeOutCubic&#39;，//关闭动画showOverlay：true，//显示覆盖层showCloseButton：false //追加关闭按钮进入siednav}）；

固定/非固定SideNav

使用类别 。固定使sideNav固定并在大屏幕上打开，并在小屏幕上隐藏为常规功能。左侧的sideNav是固定的SideNav的示例。

如果您想默认隐藏SideNav 去掉 。固定 SideNav中的类。

需要左填充使用时 。固定类

如果您打算使用此功能，则必须将内容偏移侧菜单的宽度。将填充物放置在偏移内容将要放置的位置，在我们的示例中是在页眉，主要和页脚中。

页眉，主要，页脚{padding-left：240px; } @media only屏幕和（max-width：992px）{页眉，主要，页脚{padding-left：0; }}

注意： 无需通过CSS添加所需的填充，您可以使用我们的预定义之一导航布局
