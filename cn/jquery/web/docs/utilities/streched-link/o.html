自举延伸链接

通过CSS“拉伸”嵌套的链接，使任何HTML元素或Bootstrap组件均可点击。

带有延伸链接的卡

加 .stretched-link 链接使其包含块通过点击 ：：后伪元素。在大多数情况下，这意味着具有职位：相对包含与 .stretched-link 该类是可点击的。

卡有职位：相对默认情况下在Bootstrap中，因此在这种情况下，您可以安全地添加 .stretched-link 类到卡片中的链接，而没有其他任何HTML更改。

拉伸链接不建议使用多个链接和点击目标。但是，有些位置和索引样式可以帮助您。

识别包含块

如果拉伸的链接似乎不起作用，则包含块可能是原因。以下CSS属性将使元素成为包含块：

  • 一种位置以外的价值静态的
  • 一种转变要么透视以外的价值没有
  • 一种将会改变的价值转变要么透视
  • 一种过滤以外的价值没有或一个将会改变的价值过滤 （仅适用于Firefox）
