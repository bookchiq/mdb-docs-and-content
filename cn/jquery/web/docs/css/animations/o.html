高级用法︰

当您将其与jQuery结合或添加自己的CSS规则时，可以对动画做很多其他的事情。

1。轻松使用jQuery动态添加动画：

$（&#39;＃yourElement&#39;）。addClass（&#39;animated bounceOutLeft&#39;）;

2。您还可以检测动画何时结束：

$（&#39;＃yourElement&#39;）。one（&#39;webkitAnimationEnd mozAnimationEnd MSAnimationEnd oanimationend animationend&#39;，doSomething）;

注意： jQuery.one（） 当您最多要执行事件处理程序时使用一旦 。更多信息这里 。


3。您还可以扩展jQuery以添加一个为您完成所有任务的函数：

#yourElement {-vendor-animation-duration：3s; -供应商动画延迟：2秒； -vendor-animation-iteration-count：无限； }

注意： 请确保将CSS中的“供应商”替换为适用的供应商前缀（webkit，moz等）。

滚动时显示动画

借助MDB，您可以轻松地在页面滚动上启动动画。

基本用法

步骤1： 初始化滚动脚本的脚本。

新的WOW（）。init（）;

第2步： 添加CSS类 。哇到HTML元素：在用户滚动显示该元素之前，它是不可见的。

第三步： 从中选择动画样式动画列表 ，然后将CSS类添加到HTML元素。

通过jQuery添加多个动画

如果要在整个页面中使用相同的动画，则可以使用jQuery addClass（） 一次完成。

$（“ .wow”）.addClass（“ fadeInUp”）;

可见性：隐藏；

加可见度：隐藏到哇如果您要为页面加载后立即可见的元素设置动画。因此，您避免在javascript加载后重新加载内容。

.wow {能见度：隐藏； }

选项

使用下面的自定义属性之一来更改滚动上动画的行为。

  1. 数据持续时间 ：更改动画持续时间

  2. 数据哇延迟 ：动画开始之前的延迟

  3. 数据哇偏移 ：开始动画的距离（与浏览器底部有关）

  4. 数据哇重复 ：动画重复的次数

自定义设置

  1. boxClass ：类名称，在用户滚动时显示隐藏的框

  2. 动画类 ：触发CSS动画的类名

  3. 抵销 ：定义浏览器视口底部与隐藏框顶部之间的距离。当用户滚动并到达此距离时，将显示隐藏的框。

  4. 移动 ：打开/关闭移动设备上的动画。

  5. 生活 ：不断检查页面上是否有新的动画元素

wow = new WOW（{boxClass：&#39;wow&#39;，//默认animateClass：&#39;animated&#39;，//默认偏移量：0，//默认移动设备：true，//默认live：true //默认}）wow.init（ ）;
