选项

可以传递选项数据属性或从脚本初始化jarallax时在对象中。

名称 类型 默认 描述
类型 字符串 滚动 滚动，缩放，不透明度，滚动不透明度，缩放不透明度。
速度 浮点数 0.5 视差效果速度。提供从-1.0到2.0的数字。
imgSrc 路径 空 图像网址 默认情况下使用背景图像。
imgElement dom /选择器 .jarallax-img 用作背景的图片标签。
img尺寸 字符串 封面 图像大小 如果您使用<img>标记为背景，您应该添加对象拟合值，否则使用背景尺寸价值观。
img位置 字符串 50% 50% 图像位置 如果您使用<img>标记为背景，您应该添加对象位置值，否则使用背景位置价值观。
img重复 字符串 不平铺 图片重复 仅支持背景位置价值观。
keepImg 布尔值 false 保持<img> Jarallax启动后，将其标记为默认位置。
elementInViewport ./dom 空 使用自定义DOM / jQuery元素检查视口中视差是否阻塞。更多信息在这里- 第13期 。
用zIndex 数字 -100 视差容器的Z索引。
disableParallax RegExp /功能 - 禁用特定用户代理上的视差（使用正则表达式）或使用函数返回值。图像将设置在背景上。
disableVideo RegExp /功能 - 禁用特定用户代理（使用正则表达式）或具有函数返回值的视频加载。图像将设置在背景上。
自动调整大小 布尔值 false 具实验性 当视差块的大小更改时（内部的动态内容），自动重新计算视差大小和位置。使用ResizeObserver，它与小型浏览器兼容。建议与polyfill搭配使用 https://github.com/que-etc/resize-observer-polyfill 。

在移动设备上禁用

您可以使用选项在移动设备上禁用视差效果和/或视频背景 disableParallax 和/或 disableVideo 。

Example:

jarallax（document.querySelectorAll（&#39;。jarallax&#39;），{disableParallax：/ iPad | iPhone | iPod | Android /，disableVideo：/ iPad | iPhone | iPod | Android /}）;

或使用功能。Example:

jarallax（document.querySelectorAll（&#39;。jarallax&#39;），{disableParallax：function（）{return /iPad|iPhone|iPod|Android/.test(navigator.userAgent）;}，disableVideo：function（）{返回/ iPad | iPhone | iPod | Android / .test（navigator.userAgent）;}}）;

视频选项（+支持所有默认选项）

可以传递选项数据属性或从脚本初始化jarallax时在对象中。

名称 类型 默认 描述
videoSrc 字符串 空 您可以使用Youtube，Vimeo或本地视频。你也可以使用数据属性数据jarallax视频 。
videoStartTime 浮点数 0 开始时间在秒开始视频
videoEndTime 浮点数 0 视频结束时秒的结束时间
videoVolume 浮点数 0 视频音量从0到100。
videoPlayOnlyVisible 布尔值 如果为 true，则幻灯片将没有固定位置 仅在屏幕上可见时播放视频。
