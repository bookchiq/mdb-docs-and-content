引导皮肤

Bootstrap外观是基本的配色方案，可用于提供随时使用的整页Web设计的网站。

MDB为您提供了几种基本的肤色方案，您可能会考虑将其应用到您的网站上以产生不同的效果。

要应用皮肤，您只需要将我们的皮肤类别之一添加到<body>元件。

皮肤会影响以下元素： 导航栏 ， sideNav ，页脚 ，纽扣 ，落下 ，分页 ，输入 ，轮播多项目控件 ，表格标题 ，卡头 ，微调器 ，分页 ，渐变色 ，主要，次要，默认颜色 。

添加自定义皮肤的可能性

从MDB的最新版本-4.5.2，您可以添加自定义外观。下面我们分几步说明如何执行此操作。注意：要创建自定义皮肤，最好使用 MDB Pro Gulp软件包。

添加自定义皮肤的说明：

在SCSS目录的软件包中，您可以看到 _custom-skin.scss 文件。

转到此文件并更改皮肤的名称和值 $皮肤反对您的自定义颜色并保护文件安全。之后，您将重新编译 mdb.css 新皮肤。
