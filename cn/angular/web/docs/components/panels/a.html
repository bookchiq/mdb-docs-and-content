角板-API

在本节中，您将找到有关面板及其所需模块以及可用输入，输出，方法和事件的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

// MDB Angular Pro从“ ng-uikit-pro-standard”导入{ButtonsModule，WavesModule，CardsFreeModule}
// MDB Angular Pro从“ angular-bootstrap-md”导入{ButtonsModule，WavesModule，CardsFreeModule}
汇入范例模块清单

输入

MdbCardComponent
名称 类型 默认 描述 范例
层叠 布尔值 false 允许为卡设置层叠样式。 Cascade =“ true”
更宽 布尔值 false 允许为卡片设置更多样式。 wide =“ true”
更窄 布尔值 false 允许为卡片设置较窄的样式。 arrower =“ true”
提高警觉性 布尔值 false 允许反转卡片的样式。 reverse =“ true”
黑暗 布尔值 false 允许为卡片设置深色样式。 dark =“ true”
bgColor 字符串 ' ' 允许通过使用类为卡片添加背景色。 bgColor =“ bg-primary”
borderColor 字符串 ' ' 允许通过使用类为卡添加边框颜色。 borderColor =“ border-primary”
MdbCardImageComponent
名称 类型 默认 描述 范例
src 字符串 ' ' 允许设置卡片的图片网址。 src =“ img / image1.webp”
海拔 布尔值 false 允许为卡片图像设置alt属性。 alt =“备用图片”
