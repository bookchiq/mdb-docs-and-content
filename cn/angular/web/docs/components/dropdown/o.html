Angular Bootstrap下拉列表

角度下拉菜单-Bootstrap 4和材质设计

使用Angular Bootstrap下拉菜单切换上下文叠加层以显示链接列表以及更多内容。

概述

下拉菜单是可切换的上下文叠加层，用于显示链接列表等。它们通过随附的Bootstrap下拉JavaScript插件进行了交互。它们是通过单击而不是悬停来切换的；这是故意的设计决定。

如果导航系统更为复杂，则下拉列表是必不可少的元素，使您可以将所有最相关的链接放置在导航栏中（或其他充当导航的组件-如SideNav）

除了功能强大之外，MDB下拉菜单还具有非凡的视觉吸引力。打开链接后，它们可以赋予其这种独特的材料设计效果，例如阴影，生动的色彩或迷人的波浪效果。

它们是通过单击而不是悬停来切换的；这是一个故意的设计决定。为什么？

MDB是“移动优先”框架，因此，我们避免通过悬停来混合功能元素和交互，因为它会降低移动设备用户的用户体验。

Google Material Design准则建议采用相同的方法。

可获取性

的威力咏叹调标准定义了一个实际的角色=“菜单” 小部件 ，但这特定于触发操作或功能的类似应用程序的菜单。咏叹调菜单只能包含菜单项，复选框菜单项，单选按钮菜单项，单选按钮组和子菜单。

另一方面，Bootstrap的下拉菜单被设计为通用的，适用于各种情况和标记结构。例如，可以创建包含其他输入和表单控件（例如搜索字段或登录表单）的下拉菜单。因此，Bootstrap不会期望（也不会自动添加）任何角色和咏叹调- true所需的属性咏叹调菜单。作者必须自己包括这些更具体的属性。

但是，Bootstrap确实为大多数标准键盘菜单交互添加了内置支持，例如可以逐个移动 .dropdown-item 元素使用光标键，然后使用退出键。
