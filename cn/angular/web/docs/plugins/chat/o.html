Angular Bootstrap聊天

角度聊天-Bootstrap 4和材质设计

Angular Bootstrap聊天是专门用于两个或多个用户之间基于文本的实时通信的组件。使用对话列表侧边栏进行构建。

所需样式导入

从MDB Angular 8开始，使用以下组件需要导入其他样式表。将以下导入添加到全局 styles.scss 样式表文件。

导入“〜ng-uikit-pro-standard / assets / scss / addons-pro / _chat-pro.scss”；
