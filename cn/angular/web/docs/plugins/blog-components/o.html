Angular Bootstrap Blog组件

Angular Blog组件-Bootstrap 4和材质设计

Angular Bootstrap博客组件是一组专用于构建博客或任何其他面向内容的网站的组件。

所需样式导入

从MDB Angular 8开始，使用以下组件需要导入其他样式表。将以下导入添加到全局 styles.scss 样式表文件。

导入“〜ng-uikit-pro-standard / assets / scss / core / colors”； @import“〜ng-uikit-pro-standard / assets / scss / core / variables”; @import“〜ng-uikit-pro-standard / assets / scss / core / variables-pro”; @import“〜ng-uikit-pro-standard / assets / scss / addons-pro / _blog-pro.scss”;
