拖放/可拖动

Bootstrap拖放/可拖动插件

MD Bootstrap Draggable插件是一个扩展，允许您通过单击对象并将其拖动到视口内的任何位置来移动对象。

要开始使用Draggable插件，请参见 “ API” 此页面上的标签。

自动滚屏

默认情况下，当元素移出容器边界之外时，滚动位置将自动更新。采用 [autoScroll] =“ false” 输入以禁用此功能。也可以通过以下方式更改滚动灵敏度和速度 [scrollSensitivity] 和 [scrollSpeed] 输入。
