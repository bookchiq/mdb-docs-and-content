Angular Bootstrap可排序

可按角度排序-Bootstrap 4和材料设计

Angular Bootstrap可排序插件是一个扩展，允许您使用拖放功能来移动，重新排序，排序和组织网站上的元素。

要开始使用可排序插件，请参阅 “入门” 此页面上的标签。
