Angular Bootstrap所见即所得编辑器

Angular WYSIWYG Editor插件-Bootstrap 4和Material Design

Angular Bootstrap所见即所得编辑器是一个轻巧的插件，可在您的网站上进行富文本编辑。

要开始使用可排序插件，请参阅 “入门” 此页面上的标签。

基本示例

Translations

Angular Bootstrap所见即所得编辑器附带了一个选项，允许您对编辑器中的所有文本使用自定义翻译。

自定义调色板

默认情况下，Angular Bootstrap所见即所得编辑器使用MDBootstrap的文本颜色中的调色板。如果您需要使用自定义颜色，则可以简单地自定义颜色以适合您的项目。

从“ @ angular / core”导入{Component}； @Component（{选择器：&#39;wysiwyg-colors&#39;，templateUrl：&#39;./wysiwyg-colors.component.html&#39;，styleUrls：[&#39;./wysiwyg-colors.component.scss&#39;]}）导出类WysiwygColorsComponent {options = {颜色：{红色：&#39;＃d50000&#39;，绿色：&#39;＃64dd17&#39;，黄色：&#39;＃fff176&#39;，蓝色：&#39;＃03a9f4&#39;，紫色：&#39;＃6a1b9a&#39;，白色：&#39;＃fff&#39;，黑色：&#39;＃000 &#39;}}}
