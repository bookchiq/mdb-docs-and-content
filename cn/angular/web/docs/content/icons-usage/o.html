易用性

真棒字体版本5 已完全从头开始重新编写和重新设计。

发生了什么变化

  • 3种不同的图标样式-每个图标都以纯色，常规和浅色样式引入。品牌图标分为各自的样式/类别，以方便使用。版本5具有四个前缀，可让我们轻松设置任何图标的样式：
新前缀 Icon 样式 SVG + JS文件名 Web字体文件名 可用性
很棒 字体超赞-品牌 brand.js fa-brands-400。* 自由

  • 图标名称更改-在版本5中删除了别名，每个图标都有一个正式名称，并且其中一些名称已根据新的标准和约定进行了调整。
  • 轮廓样式图标-类似地，所有具有轮廓样式（通常以-o结尾）的图标现在都具有far前缀，并且已删除了-o后缀。
