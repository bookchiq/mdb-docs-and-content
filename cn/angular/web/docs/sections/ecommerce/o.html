所需样式导入

从MDB Angular 8开始，使用以下组件需要导入其他样式表。将以下导入添加到全局 styles.scss 样式表文件。

导入“〜ng-uikit-pro-standard / assets / scss / core / colors”； @import“〜ng-uikit-pro-standard / assets / scss / core / variables”; @import“〜ng-uikit-pro-standard / assets / scss / core / variables-pro”; @import“〜ng-uikit-pro-standard / assets / scss / addons-pro / _ecommerce-pro.scss”； //仅在要使用产品示例时导入@import“〜ng-uikit-pro-standard / assets / scss / sections-pro / _pricing-pro.scss”; //仅在您想使用定价示例时导入
