高级用法

模态指令

分页选择器： [mdbModal]
导出为 mdbModal

输入

配置 类型： ModalOptions
允许通过元素属性设置模式配置

输出（事件）

（onHidden）=“ onHidden（）” 模态完成向用户隐藏后，将触发事件。
（onHide）=“ onHide（）” 调用hide实例方法后，事件立即触发。
（onShow）=“ onShow（）” 调用show实例方法时，事件立即触发。
（onShown）=“ onShown（）” 当使模式对用户可见时将触发事件。

ModalOptions

资产

背景 类型： 布尔| “静态的”
包括模式背景元素。或者，为不会关闭点击模式的背景指定静态。
ignoreBackdropClick 类型： 布尔值
忽略背景幕点击。
键盘 类型： 布尔值
当按下退出键时，关闭模态。
显示 类型： 布尔值
初始化时显示模态。
类 类型： 串
打开模态的CSS类。
动画 类型： 布尔值
切换动画。
