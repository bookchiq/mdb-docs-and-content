SideNav也称为Sidebar或简称-Side Menu，不是Bootstraps的本机组件。导航的材料设计形式很常见，这种形式很快成为现代网页设计的标准。

由于其清晰和简单，它显着提高了用户体验。它使您可以快速浏览小型应用程序和大型门户。多重链接嵌入功能使您能够实施更高级的内容分类，这在大型项目中几乎是必不可少的。

借助MDB，您可以通过使用各种诱人的侧边菜单之一轻松地在自己的项目中实现SideNav。

固定/非固定SideNav

采用 [fixed] =“ true” 和课 。固定使sideNav固定并在大屏幕上打开，并在小屏幕上隐藏为常规功能。左侧的sideNav是固定的SideNav的示例。

如果您想默认隐藏SideNav 去掉 。固定并设置 [固定] 至真正 SideNav中的类。


<mdb-side-nav #sidenav [fixed] =“ true”>

需要左填充使用时 。固定类

如果您打算使用此功能，则必须将内容偏移侧菜单的宽度。将填充物放置在偏移内容将要放置的位置，在我们的示例中是在页眉，主要和页脚中。


页眉，主要，页脚{padding-left：240px; } @media only屏幕和（max-width：992px）{页眉，主要，页脚{padding-left：0; }}

注意： 无需通过CSS添加所需的填充，您可以使用我们的预定义之一导航布局

自定义SideNav

改变颜色

您可以使用我们300种颜色中的每一种材料调色板只需添加适当的颜色分类即可。

在下面的示例中，我们添加了一个 。紫色颜色等级和 .darken-4 使4级颜色更深。


<!-SideNav菜单-> <mdb-side-nav #sidenav class =“ purple darken-4”>

浅色/深色背景

根据浅色或深色背景，您可能需要为链接和其他SideNav元素设置适当的对比度。

采用 .side-nav-light 为浅色背景。


<mdb-side-nav #sidenav“ class =” side-nav-light“
