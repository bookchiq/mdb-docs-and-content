选项

可以通过指令或提供程序传递选项。对于指令，将选项名称附加到 [...] 。

名称 类型 默认 描述
间隔 数字 5000 自动循环幻灯片之间的延迟的时间量。如果为0，则轮播不会自动循环。
没有暂停 布尔值 false 在鼠标输入时暂停旋转木马的循环，并在鼠标上恢复旋转木马的循环。
noWrap 布尔值 false 轮播是否应该连续循环或是否有硬停止。
键盘 布尔值 如果为 true，则幻灯片将没有固定位置 轮播是否应该对键盘事件做出反应。
activeSlideIndex 数字 0 当前显示的幻灯片的索引（从0开始）。您可以告诉轮播从哪一张幻灯片开始。（双向绑定[（activeSlideIndex）]）
isControls 布尔值 如果为 true，则幻灯片将没有固定位置 如果设置为false，则轮播不会显示下一个/上一个按钮。注意，[isControls]是必需的。

<mdb-carousel #carouselRef> </mdb-carousel> <a (click)=&quot;carouselRef.nextSlide()&quot;>下一步</a>
开始

从左到右循环浏览轮播项目。

暂停

阻止轮播在各个项目之间循环。

.selectSlide（数字）

将轮播循环到特定帧（从0开始，类似于数组）。

.previousSlide（）

循环到上一个项目。

.nextSlide（）

循环到下一个项目。
