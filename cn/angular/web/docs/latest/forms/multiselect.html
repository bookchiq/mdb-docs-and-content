角形引导多选

Angular Bootstrap Multiselect是一个插件，允许用户从标准Angular Bootstrap选择中打钩多个选项。

它的实现非常简单，作为交换，它带来了很多UX价值。

Angular Bootstrap Multiselect用法示例：

  • 披萨递送系统中的成分选择
  • 网上商店的笔记本电脑硬件配置
  • 机票预订定制

可用事件

在这里您可以找到由Material Multiselect触发的事件


名称 使用 描述
改变 （changed）=“ selectChanged（$ event）” 当选择选项更改时，将触发该事件。
打开 （opened）=“ selectOpened（$ event）” 打开选择时将触发该事件。
已解决 （closed）=“ selectClosed（$ event）” 选择关闭时会触发该事件。
已选择 （selected）=“ optionSelected（$ event）” 选择该项目时会触发该事件。
取消选择 （deselected）=“ optionDeselected（$ event）” 取消选中该项目时会触发该事件-仅当[multiple] =“ true”时才触发。
noOptionsFound （noOptionsFound）=“ noOptionsFound（$ event）” 当选择中没有匹配项时触发该事件-仅在[filterEnabled] =“ true”时触发。
