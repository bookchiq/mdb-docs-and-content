引导波效应

只需添加 mdbWavesEffect 到你的元素。

注意： mdbRippleRadius 现在已弃用。代替它，请使用 mdbWavesEffect 。

基本示例

点击我 

<!-带有波浪的按钮-> <a mdbBtn color=&quot;primary&quot; size=&quot;lg&quot; class=&quot;waves-light&quot; mdbWavesEffect>点击我</a> <！-面具与波浪-> <div class="view overlay hm-white-slight waves-light" mdbWavesEffect><img src="https://mdbootstrap.com/img/Photos/Horizontal/Nature/6-col/img%20(54).webp" class="img-fluid" alt="Sample image with waves effect."><a href="#"><div class="mask waves-effect"></div></a></div>

定制化

您可以将波效果颜色更改为“更浅”，并添加波光：


<!-正常-> <a mdbBtn color=&quot;primary&quot; size=&quot;lg&quot; mdbWavesEffect>点击我</a> <！-更轻-> <a mdbBtn color=&quot;primary&quot; size=&quot;lg&quot; class=&quot;waves-light&quot; mdbWavesEffect>点击我</a>

禁用波浪

您可以通过删除来禁用波浪效果 mdbWavesEffect 您元素中的指令。

API参考：

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。
