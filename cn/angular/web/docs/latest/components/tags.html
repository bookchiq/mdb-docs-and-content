用法 MDB Pro组件

添加标签


输入名称，然后按Enter键添加标签。请点击 X 删除它。

从“ @ angular / core”导入{组件}； @Component（{选择器：&#39;chips-add-example&#39;，templateUrl：&#39;chips.add.component.html&#39;，}）导出类ChipsAddComponent {hideElement：boolean = true; addtags：string [] = []; }

设置初始标签

从“ @ angular / core”导入{组件}； @Component（{选择器：&#39;chips-initial-example&#39;，templateUrl：&#39;chips.initial.component.html&#39;，}）导出类ChipsInitialComponent {hideElement：boolean = true; addtags：string [] = []; initialtags：string [] = [&#39;Tag 1&#39;，&#39;Tag 2&#39;，&#39;Tag 3&#39;]; }

Placeholer

或 从“ @ angular / core”导入{组件}； @Component（{选择器：&#39;chips-placeholder-example&#39;，templateUrl：&#39;chips.placeholder.component.html&#39;，}）导出类ChipsPlaceholderComponent {placeholderVar：字符串=“您的占位符”; }
