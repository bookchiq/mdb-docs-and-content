Angular Bootstrap画廊

角画廊-Bootstrap 4和材料设计

Angular Bootstrap画廊是具有出色的UI和UX值的组件。有了它，您可以将美丽而迷人的图像与您的网站或应用程序结合起来！

易于以灯箱画廊，多项目轮播或带字幕的幻灯片的形式构建。

Bootstrap画廊使用示例：

  • 博物馆网站上最受欢迎的绘画
  • 旅游博客中的风景照片
  • 投资组合中最好的作品
