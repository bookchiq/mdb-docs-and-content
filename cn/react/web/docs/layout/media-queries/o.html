React Bootstrap媒体查询

React Media Queries-Bootstrap 4和材料设计

React Bootstrap媒体查询是用于应用样式的查询，这些样式由设备的一般类型，浏览器特征或环境确定。

由于Bootstrap首先开发为移动设备，因此我们使用了一些媒体查询为我们的布局和界面创建合理的断点。这些断点主要基于最小视口宽度，并允许我们随着视口的变化按比例放大元素。

Bootstrap主要在源Sass文件中为布局，网格系统和组件使用以下媒体查询范围（或断点）。

媒体查询作为响应断点

//额外的小型设备（纵向电话，小于576px）//没有对xs的媒体查询，因为这是Bootstrap中的默认设置//小型设备（横向电话，576px及以上）@media（最小宽度：576px） {... } //中型设备（平板电脑，768像素及以上）@media（最小宽度：768像素）{... } //大型设备（台式机，992px及以上）@media（最小宽度：992px）{... } //大型设备（大型台式机，1200px及以上）@media（最小宽度：1200px）{... }

可通过Sass mixins进行媒体查询

由于我们用Sass编写了源CSS，因此所有媒体查询都可以通过Sass mixins获得：

// xs断点不需要媒体查询，因为它是有效的`@media（min-width：0）{...}`@include media-breakpoint-up（sm）{...} @include media-breakpoint-up （md）{...} @include media-breakpoint-up（lg）{...} @include media-breakpoint-up（xl）{... } //示例：从“最小宽度：0”开始隐藏，然后在“ sm”断点处显示。custom-class {display：none; } @include media-breakpoint-up（sm）{.custom-class {display：block; }}

响应断点的其他方向

有时我们会使用其他方向的媒体查询（给定的屏幕尺寸或更小）：

//额外的小型设备（纵向电话，小于576px）@media（最大宽度：575.98px）{... } //小型设备（横向电话，小于768px）@media（最大宽度：767.98px）{... } //中型设备（平板电脑，小于992px）@media（最大宽度：991.98px）{... } //大型设备（台式机，小于1200px）@media（最大宽度：1199.98px）{... } //超大型设备（大型台式机）//由于超大型断点的宽度没有上限，因此无需媒体查询

媒体查询Sass mixins示例

同样，这些媒体查询也可以通过Sass mixins使用：

include media-breakpoint-down（xs）{...} @include media-breakpoint-down（sm）{...} @include media-breakpoint-down（md）{...} @include media-breakpoint-down down（lg）{... } // xl断点没有媒体查询，因为它的宽度没有上限//示例：从中等断点到下的样式@include media-breakpoint-down（md）{.custom-class {display：block; }}

最小和最大媒体查询

也有媒体查询和混合，用于使用最小和最大断点宽度来定位单个屏幕尺寸段。

//额外的小型设备（纵向电话，小于576px）@media（最大宽度：575.98px）{... } //小型设备（横向电话，576像素及以上）@media（最小宽度：576像素）和（最大宽度：767.98像素）{... } //中型设备（平板电脑，768像素及以上）@media（最小宽度：768px）和（最大宽度：991.98px）{... } //大型设备（台式机，992px及以上）@media（最小宽度：992px）和（最大宽度：1199.98px）{... } //大型设备（大型台式机，1200px及以上）@media（最小宽度：1200px）{... }

最少-最多媒体查询Sass mixins

这些媒体查询也可以通过Sass mixins使用：

仅包括媒体断点（xs）{...} @包括仅媒体断点（sm）{...} @包括仅媒体断点（md）{...} @包括媒体断点- only（lg）{...} @include media-breakpoint-only（xl）{... }

媒体查询多个断点宽度

同样，媒体查询可能跨越多个断点宽度：

//示例//应用样式，从中型设备开始到超大型设备@media（最小宽度：768px）和（最大宽度：1199.98px）{... }

用于定位相同屏幕尺寸范围的Sass mixin为：

包括media-breakpoint-between（md，xl）{... }
