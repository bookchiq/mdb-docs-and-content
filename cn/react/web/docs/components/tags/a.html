标签-API

在本节中，您将找到有关“标签”组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从“ mdbreact”导入{MDBBadge，MDBChip，MDBChipsInput}；

API参考：MDBBadge属性

下表显示了MDBBadge组件的配置选项。

名称 类型 默认 描述 范例
类名 字符串 将自定义类添加到MDBBadge组件 <MDBBadge className="myClass" />
标签 字符串 SPAN 更改徽章的包装标签 <MDBBadge tag="p" />
颜色 字符串 更改徽章的颜色 <MDBBadge color="primary" />
胶囊 真假值 false 将徽章更改为药丸 <MDBBadge丸/>

API参考：MDBChip属性

下表显示了MDBChip组件的配置选项。

名称 类型 默认 描述 范例
类名 字符串 将自定义类添加到MDBChip组件 <MDBChip className="myClass" />
标签 字符串 更改MDBChip组件的大小。从中选择 lg | md | 短信 <MDBChip tag="p" />
大小 字符串 div 更改MDBChip的包装标签。 <MDBChip size="lg" />
bgColor 字符串 改变MDBChip的颜色 <MDBChip color="primary" />
文本 字符串 设置MDBChip的文本颜色 <MDBChip text="white" />
渐变 字符串 设置MDBChip的渐变样式 <MDBChip gradient="aqua" />
src 字符串 让您添加img并设置其src属性 <MDBChip src="https://mdbootstrap.com/img/Photos/Avatars/avatar-6.webp" />
海拔 字符串 定义img alt属性 <MDBChip alt="my avatar" />
关闭 真假值 将关闭图标添加到您的MDBChip，并附加到onClick事件，该事件将触发自定义handleCloseClick函数。 <MDBChip关闭/>
波 真假值 向MDBChip组件添加波浪效果。 <MDBChip波/>
处理关闭 函数 让您定义单击关闭图标时触发的自定义函数。 <MDBChip handleClose={this.myCloseFunction} />

API参考：MDBChipsInput属性

下表显示了MDBChipsInput组件的配置选项。

名称 类型 默认 描述 范例
类名 字符串 将自定义类添加到包装器组件 <MDBChipsInput className="myClass" />
标签 字符串 更改MDBChipsInput的包装标签。 <MDBChipsInput tag="p" />
handleAdd 函数 返回具有添加的MDBChipsInput的ID（索引），值（内容）和目标（html元素）的对象。 <MDBChipsInput handleAdd={(value) => console.log（value）} />
处理关闭 函数 让您定义单击MDBChip关闭图标时触发的自定义函数。 <MDBChipsInput handleClose={this.myCloseFunction} />
handleRemove 函数 返回一个ID（索引）和值（内容）为移除的MDBChipsInput的对象。 <MDBChipsInput handleRemove={(value) => console.log（value）} />
占位符 字符串 定义输入占位符 <MDBChipsInput placeholder="Custom placeholder" />
secondaryPlaceholder 字符串 定义输入辅助占位符 <MDBChipsInput secondaryPlaceholder="Custom placeholder" />
芯片尺寸 字符串 更改MDBChip的大小。从中选择 lg | md | 短信 <MDBChipsInput chipSize="lg" />
chipColor 字符串 改变MDBChip的颜色 <MDBChipsInput chipColor="cyan" />
chipText 字符串 设置MDBChip的文本颜色 <MDBChipsInput chipText="white" />
chipGradient 字符串 设置MDBChip的渐变样式 <MDBChipsInput chipGradient="aqua" />
chipGradient 字符串 设置MDBChip的渐变样式 <MDBChipsInput chipGradient="aqua" />
波 字符串 向MDBChip组件添加波浪效果。 <MDBChipsInput wave />
薯片 阵列 预定义要渲染的标签 <MDBChipsInput chips={["Tag 1", "Tag 2", "Tag 3"]} />
getValue 字符串 返回带有MDBChipsInput的实际列表的数组 <MDBChipsInput getValue={(value) => console.log（value）} />
