反应卡-Bootstrap 4和材料设计

引导卡是显示具有不同阴影，深度和悬停效果的不同元素的内容构建的组件。

除了通过卡片以外，很难想到一种更好的向用户显示内容的方法。正如您可能已经在几乎所有产品中注意到的那样，一些最大的公司（如Facebook或Google）都非常了解这一点。

卡片可为您提供清晰，干净的内容分类，以及将其呈现给用户的诱人形式。

MDB通过独特的“材料设计”功能（例如轻微的阴影，深度，级联或波浪效果）增强了Bootstrap卡。除此之外，它还通过其他可选动画，社交图标，头像和本机Bootstrap中不可用的各种其他效果来丰富它。
