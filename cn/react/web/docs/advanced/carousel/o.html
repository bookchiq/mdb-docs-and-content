React Bootstrap轮播

React Carousel-Bootstrap 4和材料设计

React Bootstrap轮播是响应式和交互式幻灯片，用于显示内容，尤其是图像和视频。

轮播是用于循环浏览一系列内容的幻灯片。它适用于一系列图像，文本或自定义标记。它还包括对上一个/下一个控件和指示器的支持。

请注意，不支持嵌套轮播，轮播通常不符合辅助功能标准。

这是仅带有幻灯片的旋转木马。请注意 .d块和 .w-100 在轮播图片上，以防止浏览器默认图片对齐。
