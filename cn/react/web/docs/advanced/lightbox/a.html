React LightBox-API

在本节中，您将找到有关LightBox组件的高级信息。您将发现需要哪些模块，配置组件的可能性以及在使用该组件时可以使用的事件和方法。

MDBLightbox导入语句

为了使用MDBLightbox组件，请确保首先导入了正确的模块。

从“ mdbreact”导入{MDBLightbox}；

MDBLightbox

下表显示了MDBLightbox组件的配置选项。

名称 类型 默认 描述 范例
图像 阵列 [] 将图像导入组件 <MDBLightbox images=[{src: 'link/to/image.webp'}] ... />
itemClassName 字符串 为图像添加自定义类 <MDBLightbox itemClassName='my-5' ... />
无保证金 真假值 false 删除图像之间的默认边距 <MDBLightbox noMargins ... />
marginSpace 号码 150 如果您打开图像，则在图像的Y轴上定义边距 <MDBLightbox marginSpace={0} ... />
lg，md，sm，size，xl，xs 字符串 定义图像的尺寸宽度 <MDBLightbox size='1' xs='1' sm='2' md='4' lg='3' xl='2' ... />
页面加载过渡 号码 400 以毫秒为单位设置过渡时间 <MDBLightbox transition={1000} ... />

MDBLightbox对象属性

要设置数据，您必须将对象导入图片属性

<MDBLightbox images=[ {src: 'link/to/image.webp', alt='...', md='3'}, {src: 'link/to/image2.webp', alt='...', md='3'} ] ... />

下表显示了图像的配置选项对象。

名称 类型 默认 描述 范例
海拔 字符串 图片 NR 将自定义alt属性设置为特定图像 {src：&#39;src / img.webp&#39;，alt =&#39;我的替代文字&#39;}
描述 字符串 在图库视图中的图像下添加描述 <{src：&#39;/img.webp&#39;，描述：&#39;灯箱项目&#39;} />
src 字符串 将图像导入组件 {src：&#39;https://mdbootstrap.com/img/Photos/Lightbox/Original/img%20(145).webp&#39;}
xs，sm，md，lg，xl，大小 字符串 定义图像的尺寸宽度 {src：“ / img.webp”，尺寸：“ 6”，md； &#39;3&#39;，lg：&#39;2&#39;}
