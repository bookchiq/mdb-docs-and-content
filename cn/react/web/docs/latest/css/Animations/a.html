小道具

以下是可用的道具：

道具名称 类型 默认 描述
类型 字符串 - 道具定义所需的动画种类
计数 号码 1 动画应具有的迭代次数
持续时间 字符串 1秒 定义动画的速度，采用由数字和单位指示符组成的字符串（ s / 女士 ）
延迟 字符串 0 设置应该经过的时间初始迭代动画，采用由数字和单位指示符组成的字符串（ s / 女士 ）
无限的 博林 false 使动画永生。注意：它会覆盖计数支柱
滚动显示图像 博林 false 设置此选项可使组件将动画推迟到视口到达其相对位置为止

动作钩

我们可以在动画的任意阶段触发事件。下面的道具具有可以在达到相关阶段时解析的功能。

Action Hook 描述
onAnimationStart 动画开始后触发
onAnimationIteration 每次动画发生时函数都会执行。要使事情仅在特定的迭代中发生（例如，前五次每次，然后每一次健康），创建一个在函数内递增的变量并定义一个简单的变量就足够了如果语句，查看正在进行的迭代次数是否满足要求（例如 if（count <6 || count％5 == 0）{yourFunction} ）
onAnimationEnd 功能应在动画的最终迭代之后发生
