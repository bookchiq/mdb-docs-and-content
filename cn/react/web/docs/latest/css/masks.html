使用

步骤1： 创建一个包装视图零件。

从“ mdbreact”导入{查看}；
<View>
    [...]
</View>

第2步： 设置图像的路径。如果要使图像具有响应性，请记住添加一个类 .img-fluid 。

<View>
    <img src="https://mdbootstrap.com/img/Photos/Others/nature-sm.webp" className="img-fluid" alt=""/>
</View>

第三步： 添加组件面具用图案覆盖图像。

从&#39;mdbreact&#39;导入{View，Mask};
<View>
    <img src="https://mdbootstrap.com/img/Photos/Others/nature-sm.webp" className="img-fluid" alt=""/>
    <Mask />
</View>

步骤4： 选择一种模式从上面的示例。

<View>
    <img src="https://mdbootstrap.com/img/Photos/Others/nature-sm.webp" className="img-fluid" alt="" />
    <Mask pattern={1} />

    </div>
</View>

步骤5： 如果要添加一些文本，可以使用一个类 .flex中心居中 .white-text 使您的文字具有更强的对比度，并使文字更清晰可见。

<View><img src="https://mdbootstrap.com/img/Photos/Others/nature-sm.webp" className="img-fluid" alt="" /><Mask pattern={1} className="flex-center"><p className="white-text"> .pattern-1 </p></Mask></View>

步骤6： （可选）添加知名波浪 / 连锁反应点击图片即可激活，只需传递一个波浪支撑视图零件。

<查看波浪> <img src="https://mdbootstrap.com/img/Photos/Others/nature-sm.webp" className="img-fluid" alt="" /><Mask pattern={1} className="flex-center"><p className="white-text"> .pattern-1 </p></Mask></View>
