筹码输入 MDB Pro组件

添加标签

芯片输入组件内部具有更复杂的逻辑。它根据状态生成芯片-输入名称，然后按Enter键以添加标签，这意味着将其添加到组件处于其状态的字符串数组中。它跟踪所渲染的芯片，因此不需要其他关闭处理方法-只需单击 X 去除碎屑。最后，它需要一个占位符道具，当筹码列表为空时，使输入呈现其内容，然后有 secondaryPlaceholder 道具，一旦有一个或多个筹码，其内容就会被应用。

<ChipsInput placeholder="+Tag" secondaryPlaceholder="Enter a tag"/>

设置初始标签

组件的状态保持芯片数据可以在渲染时更改，也可以使用筹码支柱。

<ChipsInput chips={['Tag 1', 'Tag 2', 'Tag 3']}/>
