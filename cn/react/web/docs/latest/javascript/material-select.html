使用引导选。

Bootstrap物料选择是一个表单控件，单击后会显示可折叠的多个值列表，可在表单，菜单或调查中使用。

MDB为您提供了多种选择和变体。

默认值 (选择)

默认选择不需要初始化。您只需要添加一个 .browser-default 上课选择元件。

                
<label>默认选择</label><select className="browser-default"><option value="" disabled selected>选择你的选择</option><option value="1">选项1 </option><option value="2">选项2 </option><option value="3">选项3 </option></select>
