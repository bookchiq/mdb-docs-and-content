React Parallax-API

在本节中，您将找到有关视差组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从“ mdbreact”导入{MDBParallax}；

API参考：视差属性

下表显示了MDBParallax组件的配置选项。

名称 类型 默认 描述 范例
海拔 字符串 “ MDBParallax图片” 所需道具-套装图片 alt <MDBParallax alt='MDB Parallax Image' />
类名 字符串 自定义类别 <MDBParallax className="customClass" />
高 字符串 600像素 改变视差容器的高度 <MDBParallax height="100vh" />
图片 字符串 设置视差图像的网址和。必备道具之一。 <MDBParallax image='http://yourLink.com/img' />
imgElement 字符串 .jarallax-img 用作背景的图片标签。 <MDBParallax imgElement='.your-selector' />
img位置 字符串 50% 50% 图像在容器内的位置。 <MDBParallax imgPosition='100px 100px' />
img重复 字符串 不平铺 图片重复 仅支持对象位置价值观。 <MDBParallax imgRepeat='repeat' />
img尺寸 字符串 封面 容器的图像覆盖率。 <MDBParallax imgSize='contain' />
keepImg 真假值 false Jarallax启动后，将paralax元素保留在默认位置。 <MDBParallax keepImg={true} />
速度 混合（字符串/数字） 0.5 视差效果速度。提供从-1.0到2.0的数字。 <MDBParallax speed='1.1' />
标签 字符串 div 设置视差容器标签 <MDBParallax tag='span' />
阈值 混合（字符串/数字） null null 指定启动视差效果的阈值。例如，如果传递0 0，则元素仅在滚动到视口中间后才开始移动。 <MDBParallax threshold='0 0' />
类型 字符串 滚动 视差效果 您可以使用以下五种类型之一： 滚动，缩放，不透明度，滚动不透明度，缩放不透明度。 <MDBParallax type='scroll-opacity' />
videoEndTime 号码 0 视频结束时秒的结束时间 <MDBParallax videoEndTime={0} />
disableVideoLazyLoading 真假值 false 仅在屏幕上可见时预加载视频。 <MDBParallax disableVideoLazyLoading />
disableVideoLoop 真假值 false 循环播放视频以无限播放。 <MDBParallax disableVideoLoop />
disableVideoPlayOnlyVisible 真假值 false 仅在屏幕上可见时播放视频。 <MDBParallax disableVideoPlayOnlyVisible />
视频 字符串 您可以使用Youtube，Vimeo或本地视频。 <MDBParallax video='https://player.vimeo.com/video/137857207' />
videoStartTime 号码 0 开始时间在秒开始视频 <MDBParallax videoStartTime={0} />
videoVolume 号码 0 视频音量从0到100。 <MDBParallax videoVolume='100' />
宽 字符串 100% 设置视差容器的宽度。使用时需要 keepImg 道具 <MDBParallax width='200px' />
用zIndex 号码 -100 视差容器的Z索引。 <MDBParallax zIndex={20} />
