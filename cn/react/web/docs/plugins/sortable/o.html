React Bootstrap可排序

React Sortable-Bootstrap 4和材料设计

React Bootstrap可排序插件是一个扩展，允许您使用拖放功能来移动，重新排序，排序和组织网站上的元素。

React可排序插件可在智能手机上使用，并提供了使用键盘对元素进行排序的机会。

要查看插件的所有属性和方法的列表，请参见 “ API” 此页面上的标签。
