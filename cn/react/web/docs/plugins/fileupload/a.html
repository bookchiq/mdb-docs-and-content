MDBFileupload PropTypes

下表显示了MDBFileupload组件的配置选项。

名称 类型 默认 描述 范例
messageDefault 字符串 将文件拖放到此处或单击 覆盖默认消息？ <MDBFileupload messageDefault="custom message" ... />
messageReplace 字符串 拖放或单击以替换 覆盖默认消息？ <MDBFileupload messageReplace="custom message" ... />
message删除 字符串 移除 覆盖“删除”按钮中的默认消息 <MDBFileupload messageRemove="custom message" ... />
messageError 字符串 糟糕，出了点问题。 覆盖默认错误消息 <MDBFileupload messageError="custom message" ... />
defaultFileSrc 字符串 在输入上设置默认文件 <MDBFileupload defaultFileSrc='https://mdbootstrap.com/img/Photos/Others/images/89.webp' ... />
maxFileSize 字符串 2百万 设置上传文件的最大文件大小 <MDBFileupload maxFileSize="1M" ... />
最小宽度 号码 设置上传图像的最小允许宽度 <MDBFileupload minWidth={500} ... />
maxWidth 号码 设置上传图像的最大允许宽度 <MDBFileupload maxWidth={500} ... />
minHeight 号码 设置上传图像的最小允许高度 <MDBFileupload minHeight={500} ... />
最大高度 号码 设置上传图像的最大允许高度 <MDBFileupload maxHeight={500} ... />
showRemove 真假值 如果为 true，则幻灯片将没有固定位置 隐藏删除按钮 <MDBFileupload showRemove={false} ... />
showErrors 真假值 如果为 true，则幻灯片将没有固定位置 不显示错误 <MDBFileupload showErrors={false} ... />
maxFileSizePreview 字符串 5M 设置预览图像的最大文件大小 <MDBFileupload maxFileSizePreview='1M' ... />
allowedFormats 阵列 [“人像”，“正方形”，“风景”] 允许/拒绝图片格式。从“人像”，“正方形”，“风景”中选择 <MDBFileupload allowedFormats={['portrait']} ... />
allowedFileExtensions 阵列 ['*'] 只允许某些文件扩展名 <MDBFileupload allowedFileExtensions={['jpg', 'png']} ... />
禁用 真假值 false 禁用输入 <MDBFileupload已禁用... />
集装箱高度 号码 230 设置您的MDBFileupload组件的高度[px] <MDBFileupload containerHeight={800} ... />
errorFileSize 字符串 `文件太大（最大$ {maxFileSize}）。` 覆盖默认错误消息 <MDBFileupload errorFileSize="custom message" ... />
errorFileExtension 字符串 `不允许使用该文件（仅$ {allowedFileExtensions.join（“，”）}）。` 覆盖默认错误消息 <MDBFileupload errorFileExtension="custom message" ... />
errorImageFormat 字符串 `不允许使用图像格式（仅$ {allowedFormats}）。` 覆盖默认错误消息 <MDBFileupload errorImageFormat="custom message" ... />
errorMaxHeight 字符串 `图片高度过大（$ {maxHeight} px max）。` 覆盖默认错误消息 <MDBFileupload errorMaxHeight="custom message" ... />
errorMinHeight 字符串 `图像高度太小（$ {minHeight} px min）。` 覆盖默认错误消息 <MDBFileupload errorMinHeight="custom message" ... />
errorMaxWidth 字符串 `图像宽度太大（$ {maxWidth} px max）。` 覆盖默认错误消息 <MDBFileupload errorMaxWidth="custom message" ... />
errorMinWidth 字符串 `图像宽度太小（$ {minWidth} px min）。` 覆盖默认错误消息 <MDBFileupload errorMinWidth="custom message" ... />
