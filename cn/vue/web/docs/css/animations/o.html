滚动时显示动画

借助MDB，您可以轻松地在页面滚动上启动动画-只要元素在屏幕上完全可见

基本用法

步骤1： 从&#39;mdbvue&#39;导入animateOnScroll指令

第2步： 将animateOnScroll添加到指令

第三步： 从中选择动画样式动画列表并将指令设置为其名称：

步骤4： 通过将对象传递给v-animateOnScroll指令，根据需要自定义动画。除了动画课您可以指定延迟或位置-延迟需要时间作为参数（以毫秒为单位），而位置是动画开始前用户必须滚动的视口高度的另外百分比。
