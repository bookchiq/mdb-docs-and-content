渲染内容

所需的唯一属性 mdbCarousel 工作是项，它接受两种数据类型： 数组和数 。当给定一个数组时，它将根据传递的数据渲染一个简单的图像/视频轮播，而数字将解析为此数量的命名插槽。插槽仅以索引命名（从1开始）。每张幻灯片必须参考相应的插槽包装在模板标签中。

API参考：属性

名称 类型 默认 描述 范例
项目-必填 数组，数字 指定项目的数量（如果是数组，则为内容） <mdb-carousel :items="3">
间隔 号码 如果给定值，则该组件将在此时间后更改滑动（以毫秒为单位） <mdb-carousel :items="3" :interval="8000">
标签 字符串 div 指定组件的包装标签 <mdb-carousel :items="3" tag="span">
幻灯片 真假值 false 将动画更改为滑动效果 <mdb-carousel :items="3" slide>
控制 真假值 false 如果设置为true，将显示控制箭头 <mdb-carousel :items="3" controlls>
控制图标 阵列 设置控件的自定义图标 <mdb-carousel :items="3" :controllIcons="['fas fa-arrow-circle-left fa-2x text-white', 'fas fa-arrow-circle-right fa-2x text-white']">
指标 真假值 false 如果设置为true，将显示指示器 <mdb-carousel :items="3" indicators>
多个 真假值 false 将轮播的样式更改为多项目 <mdb-carousel :items="3" multi>
1 真假值 false 使多项目轮播幻灯片每次滑动一次 <mdb-carousel :items="3" multi one>
缩略图 真假值 false 将轮播的样式更改为缩略图 <mdb-carousel :items="3" thumbnails>
评价 真假值 false 将轮播的样式更改为见证 <mdb-carousel :items="3" testimonial>
摸 真假值 false 向轮播中添加向右/向左滑动事件 <mdb-carousel :items="3" touch>
浮动 真假值 如果为 true，则幻灯片将没有固定位置 将控件的样式设置为浮动 <mdb-carousel :items="3" controlls :floating="false">
leftIcon 字符串 雪佛龙左 更改左控制按钮的图标 <mdb-carousel :items="3" controlls leftIcon="paper-plane">
rightIcon 字符串 雪佛龙左 更改右控制按钮的图标 <mdb-carousel :items="3" controlls rightIcon="paper-plane">
导航类 字符串 向控件添加新类 <mdb-carousel :items="3" controlls navClass="custom_class">
navColor 字符串 首選 更改控件的颜色 <mdb-carousel :items="3" controlls navColor="secondary">
指示灯颜色 字符串 首選 更改指示器的颜色 <mdb-carousel :items="3" controlls indicatorsColor="secondary">
thumbnailWidth 号码 100 更改缩略图的宽度 <mdb-carousel :items="3" thumbnails :thumbnailsWidth="50">

-属性仅适用于PRO用户。
