SideNav-API

在本节中，您将找到有关SideNav组件的高级信息。您将了解此组件中需要哪些模块，配置该组件有哪些可能性以及可以使用哪些事件和方法来使用它。

导入账单

从&#39;mdbvue&#39;导入{mdbSideNav，mdbSideNavNav，mdbSideNavCat，mdbSideNavItem};

API参考：mdbSideNav属性

名称 类型 默认 描述 范例
标签 字符串 div 更改side-nav的包装标签 <mdb-side-nav tag="ul" />
sideNavClass 字符串 覆盖或扩展应用于组件的样式。 <mdb-side-nav sideNavClass =“ example-class” ... >
固定 真假值 false 将sidenav更改为固定-当窗口宽于其断点（1440px）时始终可见 <mdb-side-nav固定... >
hide 真假值 false 将sidenav更改为hidden-始终处于隐藏状态，但可以通过任何切换按钮打开 <mdb-side-nav隐藏... >
还可以 真假值 false 将sidenav位置更改为右 <mdb-side-nav对... >
logo 字符串 启用徽标包装并指向其src属性 <mdb-side-nav logo =“ https://mdbootstrap.com/img/logo/mdb-transaprent-noshadows.webp” ... >
logoClass 字符串 自定义徽标类 <mdb-side-nav logoClass =“ example-class” ... >
logoAlt 字符串 设置徽标图像Alt Atrribute <mdb-side-nav logoAlt =“徽标” ... >
href 字符串 '#' 自定义徽标包装的href <mdb-side-nav href =“ index” ... >
到 字符串 应用“到”道具以使用路由器链接 <mdb-side-nav to =“ / index” ... >
breakWidth 号码 1440 固定sideNav的断点 <mdb-side-nav：breakWidth =“ 770” ... >
颜色 字符串 sideNav的背景色 <mdb-side-nav color =“ indigo” ... >
暗波 真假值 false 改变波浪成黑暗的版本 <mdb-side-nav darkWaves ... >
面具 字符串 为sideNav组件添加蒙版 <mdb-side-nav mask =“ strong” ... >
logoSn 真假值 false 更改徽标包装器类 <mdb-side-nav logoSn ... >
较小 真假值 false 启用辅助菜单的苗条菜单选项 <mdb-side-nav苗条... >
logoRound 真假值 false 将圆形蒙版应用于徽标 <mdb-side-nav logo圆形... >
名称 字符串 在sidenav的精简版中，此属性指定徽标旁边显示的文本-最小化sidenav时将不可见 <mdb-side-nav name =“ Anna Deynah” ... >
已崩溃 真假值 false 指示sidenav是否已最小化。该属性应绑定到与已崩溃来自的财产 mdb-side-nav-item 零件 <mdb-side-nav：isCollapsed =“ data” ... >

API参考：mdbSideNavCat属性

名称 类型 默认 描述 范例

显示 真假值 如果为 true，则幻灯片将没有固定位置 默认情况下打开类别列表 <mdb-side-nav-cat show />
較遠的 / 较远的 (jiào yuǎn de) 真假值 false 定期更改图标的样式 <mdb-side-nav icon =“ ...”远... >
定期 真假值 false 定期更改图标的样式 <mdb-side-nav icon =“ ...”常规... >
Fal 真假值 false 更改灯光的图标样式 <mdb-side-nav icon =“ ...” fal ... >
明亮 真假值 false 更改灯光的图标样式 <mdb-side-nav icon =“ ...”点亮... >
很棒 真假值 false 更改品牌的图标样式 <mdb-side-nav icon =“ ...”晶圆厂... >
品牌 真假值 false 更改品牌的图标样式 <mdb-side-nav icon =“ ...”品牌... >
時尚潮流 真假值 false 更改图标样式的双色调 <mdb-side-nav icon =“ ...”流行... >
双色调 真假值 false 更改图标样式的双色调 <mdb-side-nav icon =“ ...”双色调... >

API参考：mdbSideNavItem属性

名称 类型 默认 描述 范例
标签 字符串 li 更改side-nav-item的包装标签 <mdb-side-nav-item tag="ul" />
href 字符串 '#' 自定义href属性 <mdb-side-nav-item href =“ index” ... >
到 字符串 应用“到”道具以使用路由器链接 <mdb-side-nav-item to =“ / index” ... >
图标 字符串 定义项目的图标 <mdb-side-nav-item icon="user" />
标题 真假值 false 使项目表现为类别元素（标题） <mdb-side-nav-item标头/>
切换按钮 事件 此事件的处理程序应切换已崩溃的性质 mdb-side-nav 和 mdb-side-nav-item <mdb-side-nav-item @ toggle =“ toggleMenu” />
已崩溃 真假值 false 指示sidenav是否已最小化。该属性应绑定到与已崩溃来自的财产 mdb-side-nav 组件并由 @切换处理程序 <mdb-side-nav：isCollapsed =“ data” ... >
较小 字符串 此属性允许提供较短的字符串，一旦sidenav折叠，该字符串将显示。如果您想显示一个图标，请使用 slimIcon属性 <mdb-side-nav slim =“ info” ... >
slimIcon 字符串 指定在sidenav折叠时是否应使用图标替换文本。 <mdb-side-nav slimIcon =“信封” ... >
切换 真假值 false 允许使用给定 mdb-side-nav-item 作为苗条的sidenav切换器 <mdb-side-nav切换器... >
openIcon 字符串 指定将切换器最小化后应显示的图标。这样，可以改变箭头的方向。 <mdb-side-nav切换器图标=“ angle-double-right” openIcon =“ angle-double-left” ... >
較遠的 / 较远的 (jiào yuǎn de) 真假值 false 定期更改图标的样式 <mdb-side-nav icon =“ ...”远... >
定期 真假值 false 定期更改图标的样式 <mdb-side-nav icon =“ ...”常规... >
Fal 真假值 false 更改灯光的图标样式 <mdb-side-nav icon =“ ...” fal ... >
明亮 真假值 false 更改灯光的图标样式 <mdb-side-nav icon =“ ...”点亮... >
很棒 真假值 false 更改品牌的图标样式 <mdb-side-nav icon =“ ...”晶圆厂... >
品牌 真假值 false 更改品牌的图标样式 <mdb-side-nav icon =“ ...”品牌... >
時尚潮流 真假值 false 更改图标样式的双色调 <mdb-side-nav icon =“ ...”流行... >
双色调 真假值 false 更改图标样式的双色调 <mdb-side-nav icon =“ ...”双色调... >
苗条的 真假值 false 定期更改slimIcon的样式 <mdb-side-nav slimIcon =“ ...” slimFar ... >
苗条的 真假值 false 改变灯的slimIcon风格 <mdb-side-nav slimIcon =“ ...” slimFal ... >
slimFad 真假值 false 更改slimIcon的双色调样式 <mdb-side-nav slimIcon =“ ...” slimFad ... >
薄板 真假值 false 改变slimIcon品牌的风格 <mdb-side-nav slimIcon =“ ...” slimFab ... >
