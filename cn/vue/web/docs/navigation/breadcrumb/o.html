更换隔板

分隔符通过以下方式自动添加到CSS中 ：：之前和内容 。他们可以通过改变来改变 $面包屑分隔符 。的引用需要函数来生成字符串周围的引号，因此，如果需要 > 作为分隔符，您可以使用以下命令：

$ breadcrumb-divider：quote（“>”）;

也可以使用 base64嵌入式SVG图标 ：

$面包屑分压器：URL（数据：图像/ SVG + xml的; BASE64，PHN2ZyB4bWxucz0iaHR0cDovL3d3dy53My5vcmcvMjAwMC9zdmciIHdpZHRoPSI4IiBoZWlnaHQ9IjgiPjxwYXRoIGQ9Ik0yLjUgMEwxIDEuNSAzLjUgNCAxIDYuNSAyLjUgOGw0LTQtNC00eiIgZmlsbD0iY3VycmVudENvbG9yIi8 + PC9zdmc +）;

分隔符可以通过设置来移除 $面包屑分隔符至没有 ：

$ breadcrumb-divider：无；

可获取性

由于面包屑可提供导航，因此最好添加有意义的标签，例如 aria-label =“ breadcrumb” 描述在<nav>元素，以及应用 aria-current =“ page” 到集合的最后一项，以表示它代表当前页面。

有关更多信息，请参见面包屑模式的WAI-ARIA创作实践 。
