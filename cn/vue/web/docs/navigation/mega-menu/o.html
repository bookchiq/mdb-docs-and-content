Vue Mega菜单

Vue Mega菜单-Bootstrap 4和材料设计

Vue mega菜单是导航组件，可增强标准导航栏功能。扩展下拉菜单包含图像和分类链接。

它可能包含较小版本的站点地图和图像。

您可以将其分为几类，从而增强网站的用户体验。

Vue巨型菜单使用示例：

  • 多分类
  • 跨国公司页面
  • 菜单中带有目的地图像的旅行页面
