Vue砌体-API

在本节中，您将找到有关砌体组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从&#39;mdbvue&#39;导入{mdbMasonry，mdbMasonryItem};

mdbMasonry属性

名称 类型 默认 描述 范例
标签 字符串 div 更改砌体包装标签 <mdb-masonry tag="p">
水平 真假值 false 将布局模式更改为水平 <水平mdb砌体>
弹性框 真假值 false 将布局模式更改为flexbox <mdb-masonry flexbox>
最大高度 字串号 自动 更改包装器的max-height属性。如果给出数字，则返回像素值。 <mdb-masonry :maxHeight="1000">
numCols 号码 定义基本布局中的列数。 <mdb-masonry :numCols="3">
相应迅速 真假值 false 在小于1200像素的屏幕上，石工将重新计算为两列布局，在移动显示器上将重新计算为一列。 <响应mdb的砌体>

mdbMasonryItem属性

.webp
名称 类型 默认 描述 范例
标签 字符串 div 更改砖石物品的标签 <mdb-masonry-item tag="p">
顺序 字串号 '' 更改砌体的顺序 <mdb-masonry-item :order="2">
itemStyle 对象 更改砌体项目的样式 <mdb-masonry-item :itemStyle="{'width': '413px'}">
src 字符串 将src属性设置为传递的字符串，将砖石项目的内容更改为图像。<mdb-masonry-item src =&#39;.. / assets / img.webp&#39;“
