自举延伸链接

通过CSS“拉伸”嵌套链接，使任何VUE组件或HTML元素都可单击。

将此组件用作路由器链接 ，设置标签属性： tag =“路由器链接”

带有延伸链接的卡

采用 mdb-stretched-link 并使其包含块可点击的。在大多数情况下，这意味着具有职位：相对包含一个 mdb-stretched-link 将可点击。

卡有职位：相对默认情况下，因此在这种情况下，您可以安全地嵌套 mdb-stretched-link 没有任何其他变化。

不建议在同一包含块中使用多个拉伸的链接和点击目标。

识别包含块

如果拉伸的链接似乎不起作用，则包含块可能是原因。以下CSS属性将使元素成为包含块：

  • 一种位置以外的价值静态的
  • 一种转变要么透视以外的价值没有
  • 一种将会改变的价值转变要么透视
  • 一种过滤以外的价值没有或一个将会改变的价值过滤 （仅适用于Firefox）
