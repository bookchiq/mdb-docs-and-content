“关闭”模式事件

您可能已经注意到，模态会再发出一个事件，即关 。每当触发任何模式关闭机制（覆盖或关闭按钮单击或用户定义的一种）并用于处理基于状态（布尔数据对象属性）的模式生命周期时，就会触发该事件。
