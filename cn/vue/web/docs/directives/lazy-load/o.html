Vue引导程序延迟加载 MDB Pro组件

Vue延迟加载-Bootstrap 4和材料设计

的延迟加载是一个自定义指令，仅当媒体在屏幕上可见时才加载媒体。

实时预览

基本用法

步骤1： 从&#39;mdbvue&#39;导入指令

第2步： 将mdbLazyLoad添加到指令宾语

第三步： 将指令附加到必须加载的元素的包装器。 SRC 属性应添加为图像或视频数据源 。采用装货定义预加载图像的键。

选项

名称 类型 默认 描述 范例
选择器 字符串 未成熟粒细胞 指定要在包装器中加载的元素选择器。可用值为“ img”，“视频”或“此”。&#39;this&#39;对于使用没有任何包装的延迟加载很有用。 <div v-mdb-lazy-load =“ {选择器：&#39;img&#39;}”> </div>
偏移量 号码 0 指定必须加载元素时从底部开始的偏移量 <div v-mdb-lazy-load =“ {offset：20}”> </div>
动画 字符串 淡入 指定用于加载元素的动画 <div v-mdb-lazy-load =“ {动画：&#39;tada&#39;}”> </div>
錯誤 字符串 &#39;https://placehold.it/600x400?text=正在加载&#39; 指定发生错误时要加载的图像 <div v-mdb-lazy-load =“ {错误：&#39;https://placehold.it/600x400?text=Loading&#39;}”> </div>
载入中 字符串 &#39;https://placehold.it/600x400?text=正在加载&#39; 指定预加载的图像 <div v-mdb-lazy-load =“ {loading：&#39;https://placehold.it/600x400?text=Loading&#39;}”> </div>
