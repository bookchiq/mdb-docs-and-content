虚拟插件可创建占位符内容，以加速构建和测试前端 模板。

注意：阅读 API 标签以找到所有可用选项，然后 高级定制

倒数计时插件会创建倒数计时器，指示到给定日期的剩余时间。

基本例子

通过向容器元素添加 data-mdb-countdown 创建默认的倒计时 倒计时到期日期值。添加 countdown-unit 类和一个属性 data-mdb-，其时间单位指向子元素，以指示应使用的时间单位 显示在倒数计时内。

间隔

创建间隔，该间隔将在每次初始倒计时结束时重置计时器。

标签

通过在 data-mdb-countdown-label 中添加文本，为每个时间单位创建标签 你的选择。

分隔器

将 data-mdb-countdown-separator 添加到容器元素以插入分隔符 每个时间单位之间。

倒数栏位置不会显示分隔符

造型

倒数位置

使用更改倒计时的默认水平位置 data-mdb-countdown-position =“ vertical”

标签位置

使用更改倒数标签的默认垂直位置 data-mdb-countdown-label-position =“水平”

字体大小

使用更改默认时间单位文本大小（4rem） data-mdb-countdown-text-size 。

自订课程

向时间单位值添加自定义类，并使用 data-mdb-countdown-text-style 和 data-mdb-countdown-label-style
