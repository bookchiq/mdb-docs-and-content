MDB Pro component

MDB treeview插件用于显示从根项开始的层次结构信息 并继续其子孙及其子女。除根以外的每个项目都有一个 父母，可以有孩子。

父节点是层次结构中较高的节点，子节点是层次结构中较低的节点。 兄弟姐妹是具有相同父代的项。项目可以展开和折叠。

注意：阅读 API 标签以找到所有可用选项，然后 高级定制

将 .treeview 类与列表一起添加到容器中以初始化组件 结构体。如果您想嵌套列表，请在其中包裹 li 标记的文本 a 标记，并在其后定义其他 ul 。将 .show 类添加到 您要从头开始扩展的列表。

基本示例-JavaScript

您可以仅使用JavaScript定义树的整体结构。数据中有一些道具 您可以用来自定义它：

  • 名称-为项目定义文本
  • children -定义一个嵌套列表及其子级
  • icon -为嵌套列表定义自定义展开图标
  • show -定义列表是否从头开始扩展
  • disabled -定义列表项是否被禁用
 const jsTreeview = document.getElementById('sample-element'); const jsInstance = new mdb.Treeview(jsTreeview, { structure: [ {name: 'One'}, {name: 'Two'}, {name: 'Three', children: [ {name: 'Second-one'}, {name: 'Second-two'}, {name: 'Second-three', children: [ {name: 'Third-one', children: [ {name: 'Fourth-one'}, {name: 'Fourth-two'}, {name: 'Fourth-three'} ]}, {name: 'Third-two'}, {name: 'Third-three', children: [ {name: 'Fourth-one'}, {name: 'Fourth-two'}, {name: 'Fourth-three'} ]} ]} ]} ], });

自定义图标-JavaScript

通过将图标代码粘贴到列表元素中，将图标添加到列表元素 name 属性。如果要更改嵌套列表中的可旋转图标，请使用 改为使用 icon 属性。

 const customIconsTreeview = document.getElementById('custom-icons-js'); const customIconsInstance = new mdb.Treeview(customIconsTreeview, { structure: [ { name: ' One' }, { name: ' Two' }, { name: ' Three', show: true, icon: '', children: [ { name: ' Second-one' }, { name: ' Second-two' }, { name: ' Second-three', icon: '', children: [ { name: ' Third-one', icon: '', children: [ { name: ' Fourth-one' }, { name: ' Fourth-two' }, { name: ' Fourth-three' }, ], }, { name: ' Third-two' }, { name: ' Third-three', icon: '', children: [ { name: ' Fourth-one' }, { name: ' Fourth-two' }, { name: ' Fourth-three' }, ], }, ], }, ], }, ], });
