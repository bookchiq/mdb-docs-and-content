MDB Pro component

Parallax是一个为图像添加滚动效果动画的插件。

注意：阅读 API 标签以找到所有可用选项，然后 高级定制

基本例子

要初始化视差效果，只需将 parallax 类添加到任何元素，然后 提供 data-mdb-image-src 属性以及图像的路径

方向

使用 data-mdb-direction 属性更改视差滚动方向

结合水平和垂直方向

延迟

使用 data-mdb-delay 属性更改滚动的延迟

规模

使用 data-mdb-scale 属性更改图像的比例

滚动过渡

使用 data-mdb-transition 更改默认滚动过渡计时功能

最大过渡

使用 data-mdb-max-transiton 属性更改图像的最大滚动值。 给出0-100范围内的值

最大图像高度

使用 data-mdb-max-height 设置最大视差容器高度

图像对齐

默认情况下，图像的水平和垂直对齐设置为居中。改变水平 与 data-mdb-horizontal-alignment 对齐并与 data-mdb-vertical-alignment （垂直对齐仅适用于 data-mdb-max-height 设置）

Image horizontally aligned

图像垂直对齐

容器溢出

使用 data-mdb-overflow 使视差溢出其容器

视差与内容

通过在类中添加 div ，将任何内容放入视差中 parallax-content 和您的内容在视差容器中
