Una tarjeta es un contenedor de contenido flexible y extensible. Incluye opciones para encabezados y pies de página, una amplia variedad de contenido, colores de fondo contextuales y una pantalla potente opciones.

Ejemplos básicos

Nota: a continuación, se muestran ejemplos de tarjetas básicas con contenido mixto y ancho fijo . Las tarjetas no tienen un ancho fijo para comenzar, por lo que se llenarán naturalmente el ancho completo de su elemento padre.

Para controlar el ancho de la tarjeta, colóquela en la cuadrícula, use las utilidades de tamaño o configure el ancho en línea. Puedes aprender más en el sección de tallas .

Transmitir significado a las tecnologías de asistencia:
El uso del color para agregar significado solo proporciona una indicación visual, que no se transmitirá a usuarios de tecnologías de asistencia, como lectores de pantalla. Asegúrese de que la información indicada por el color es obvio por el contenido mismo (por ejemplo, el texto visible), o está incluido a través de medios alternativos, como texto adicional oculto con el .sr-only clase.
