Basic example

La forma más sencilla de implementar la animación es utilizar atributos de datos. En el siguiente ejemplo, usa el icono <i class="fas fa-car-side fa-3x"></i> y agrega los atributos data-toggle="animation" data-reset="true" data-animation="slide-out-right" para darle animación al hacer clic.

data-toggle = "animation" es un atributo obligatorio para cada animación.

data-reset = "true" le permite decidir si la animación se puede repetir

data-animation = "slide-right" le permite especificar qué animación se aplica a la elemento. En la sección de demostración anterior, puede encontrar animaciones.

Haga clic en el coche para iniciar la animación.

Launch options

Hay varias opciones para iniciar la animación.

On click

La animación al hacer clic es una opción de inicio predeterminada , por lo que no requiere cualquier atributo de datos.

En vuelo estacionario

Utilice data-start = "onHover" para iniciar la animación al pasar el mouse.

En desplazamiento

Utilice data-start = "onScroll" para iniciar la animación cuando se desplaza por la página y llegar al elemento.

Tenga en cuenta que la animación se iniciará solo una vez , incluso si la alcanza. al desplazarse varias veces.

Repetir animación en scroll

Si desea iniciar la animación cada vez que se alcanza cuando se desplaza, use data-animate-on-scroll = "repeat" .
